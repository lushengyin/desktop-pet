- 实现 Electron + React + TypeScript 桌面宠物应用：透明宠物窗口、拖动持久化、菜单栏、深色设置面板、噜噜素材、自定义宠物导入/切换、配置恢复、烟测和本地 macOS 打包。

- 修复宠物大小调整时桌面宠物被窗口裁切导致形象缺失的问题，宠物精灵现在随窗口整体缩放。
- 修复显示页“重置位置”无效问题，设置应用流程现在尊重保存的 petPosition，并在重置时直接移动宠物窗口。

- 为设置窗口左侧导航添加独立滚动区域，窗口高度不足时也能访问底部选项卡。

- 新增“当前动作”功能：可在宠物页选择默认动作并立即预览，动作会持久化。
- 新增内置宠物“咕咕嘎嘎”，并将内置宠物加载改为自动扫描 public/pets/*。
- 新增宠物云朵提示：头顶漂浮云朵跟随宠物浮动，循环播放文案，并支持用户自定义文案与开关。

- 优化宠物云朵布局：减少遮挡脸部、放宽文字显示区域，并新增云朵上下/左右偏移滑杆供用户微调位置。
- 修复宠物云朵被宠物窗口边界裁切的问题：桌面宠物改为宠物锚点加云朵安全区布局，云朵相对宠物头部定位，并保留用户可调偏移。
- 拖动宠物时根据拖动方向切换奔跑动画：往右拖向右跑，往左拖向左跑。
- 为支持未来多宠物，将 UI 中绑定单一宠物名的通用文案统一替换为“桌边小伴 / Desk Buddy”。
- 修复云朵气泡随宠物体型变大而离头部越来越远的问题，并调整气泡箭头更贴近头部方向。
- 调整云朵气泡与宠物的距离：更贴近头部且不遮挡。
- 云朵气泡距离再次收近：减少气泡与宠物之间的空隙。
- 云朵气泡宽度改为随文字动态调整：短文案不再显示过长气泡。
- 修复短文案时云朵气泡漂远：改为以宠物头部为基准锚定气泡位置。
- 修复从 Mission Control 切换回来时设置窗口无法聚焦的问题。
- 应用支持按宠物自定义动作下拉文案，并优化宠物预览底色。
- 新增自定义宠物管理能力：可在宠物库删除导入的自定义宠物，内置宠物不可删除。
- 新增自定义宠物动作帧数配置，宠物资源可声明每个动作实际播放帧数。
- 新增桌面宠物透明区域鼠标穿透，宠物可拖动区域之外的窗口空白不再拦截下层点击。

## 2026-05-06

- 规划新增智能陪伴与角色记忆模块：角色主动说话、聊天互动、长期记忆、每角色独立记忆空间、手动气泡兼容和专门配置入口。
- 实现智能陪伴第一版：新增陪伴设置页、桌面快捷聊天框、OpenAI-compatible 回复服务抽象、本地回退回复、每角色独立聊天/记忆存储、主动说话调度和手动/AI/混合气泡模式。
- 智能陪伴模型配置新增 DeepSeek 和 GLM 预设，切换服务时自动填入推荐 API Base URL 和默认模型，同时保留手动修改能力。
- 主动频率从固定滑杆改为秒级数字输入框，最低支持 5 秒，方便用户自定义短间隔测试角色主动说话。
- 精简模型服务配置区说明文案，并新增测试连接按钮，可用当前供应商、Base URL、模型和 API Key 发起真实连接测试。
- 优化智能陪伴记忆与频率体验：主动频率输入不再设置 5 秒下限；角色记忆改为固定数量标签摘要，不再随主动回复无限增长。
- 修复主动频率数字框清空后自动回填旧数值的问题，输入框现在会保留空白直到用户主动输入有效数字。
- 新增每角色独立的角色设定配置，可设置称呼、性格、语气、关系感和说话风格，并注入智能陪伴回复 prompt。
- 优化陪伴设置交互：角色设定改为本地草稿加手动保存，避免输入跳动；新增设置页快捷聊天区，让用户可直接测试角色对话。
- 优化桌面宠物快捷聊天入口，将按钮移到宠物右上侧并改为“聊”标识，同时修正透明穿透交互区域。
- 移除桌面宠物旁的蓝色聊天入口，保留设置页陪伴模块中的快捷聊天能力，避免破坏宠物外观。
- 将云朵气泡尾巴菱形升级为桌面快捷聊天入口，点击后弹出会话框，可直接与当前角色聊天。
- 修复 macOS 菜单栏图标不可见问题，Tray 图标改为 template image 并保留系统图标兜底。
- 将菜单栏 tray.png 替换为确定可见的 macOS template 专用黑白图标，避免彩色透明宠物图在菜单栏渲染不可见。
- 重构宠物配置页信息架构：将陪伴和行为合并进宠物模块，内部用形象动画、互动行为、智能陪伴三个分段页组织，并优化陪伴页排版密度和层级。
- 修复智能陪伴页布局问题：角色设定输入框不再互相重叠，记忆与模型配置改为单列自适应，避免 API 配置区域显示不全。
- 修复宠物页云朵偏移卡片布局，滑杆在紧凑卡片内自适应铺满，标题不再被挤成竖排。
- 优化主动频率数字输入框外观，隐藏原生上下箭头，让控件更贴合深色卡片风格。
- 将宠物库从大卡片网格改为可搜索、可筛选的紧凑列表，更适合未来大量宠物管理。
- 优化桌面聊天气泡交互：气泡 hover 时显示“回复”按钮，点击可打开聊天框；每个角色聊天记录上限收紧为 40 条，避免长期占用膨胀。
- 桌面聊天框新增收起按钮，可随时关闭聊天面板恢复普通气泡显示。
- 模型服务支持多个配置，可新增、删除并切换当前模型；旧版单模型配置会自动迁移为默认配置。
